# Dice Corner Duel

Folder structure:
- `mobile-index.html` — main page
- `assets/css/mobile-style.css` — all styles
- `assets/js/game.js` — full game logic

## How to run
Open `index.html` in a browser.

## Current features
- 8x8 board
- Player vs Bot
- Random blocked tiles with path safety
- Round and match system
- Roll Dice button animation
- Winner popup and action log
